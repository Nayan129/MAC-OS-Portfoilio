# macOS Portfolio

An interactive macOS-inspired portfolio built with React.  
This project simulates a desktop-like experience inside the browser with draggable windows, terminal commands, and dock-based navigation.

## Live Demo

https://mac-os-portfoilio-virid.vercel.app/

---

## Features

- macOS-style desktop interface
- Interactive dock with app icons
- Draggable and closable windows
- Terminal-style portfolio with commands
- GitHub projects viewer
- Resume viewer inside a window
- Notes application
- Spotify integration
- Responsive layout

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Nayan129/MAC-OS-Portfoilio.git
2. Navigate into the project
cd MAC-OS-Portfoilio
3. Install dependencies
npm install
4. Run development server
npm run dev
5. Build for production
npm run build
Terminal Commands
Inside the portfolio terminal, you can use:

about      → Information about me
projects   → List of main projects
skills     → Technical skills
experience → Work experience
contact    → Contact details
github     → Featured GitHub projects
help       → Show all commands
```
